📋 Общие сведения

Настоящая политика конфиденциальности описывает, как приложение «Khram Aikhal Cards» обрабатывает данные при использовании Pinterest API для публикации праздничных православных открыток.

Приложение является некоммерческим и создано для распространения бесплатных открыток к православным праздникам.

🔒 Какие данные мы собираем

Приложение не собирает личные данные пользователей.

  • Не запрашиваем имена, адреса, email или номера телефонов
  • Не используем cookies для отслеживания
  • Не передаём данные третьим лицам
  • Не храним данные об активности посетителей сайта

📌 Использование Pinterest API

Приложение использует Pinterest API исключительно для:

  • Создания тематических досок с православными открытками
  • Публикации изображений открыток к церковным праздникам
  • Добавления описаний и ссылок на сайт храма
Все публикуемые материалы — открытки к праздникам — распространяются бесплатно и находятся в открытом доступе.

🖼️ Публикуемые материалы

Через Pinterest API публикуются только:

  • Изображения православных праздничных открыток
  • Текстовые описания с поздравлениями и хэштегами
  • Ссылки на сайт храма: khram-aikhal.github.io
